# 🎨 拾光涂鸦 (Glow Scribble)

> **✨ 记录闪光时刻 • 收集生活糖果 🍬**

**拾光涂鸦 (Glow Scribble)** 是一款基于 **Vue 3 + Express** 全栈开发的多巴胺风格个人生活站。项目采用极具视觉冲击力的 **Neubrutalism Pop Art (新粗野波普艺术)** 设计语言，用高饱和度的色彩和大胆的粗边框治愈你的每一天。

## 🌟 核心功能 (Features)

*   📒 **心情日记 (Mood Diary)**
    *   记录当下的情绪起伏，支持 5 种心情状态（开心、平静、难过、生气、兴奋）。
    *   可视化心情色条，回顾每一段回忆。
*   ✅ **愿望清单 (Wishlist / Todo)**
    *   规划今日任务，打卡小目标。
    *   多巴胺风格的完成动画，让打勾变得上瘾。
*   🍅 **番茄专注 (Pomodoro Focus)**
    *   内置 25 分钟专注 + 5 分钟休息模式。
    *   全屏沉浸式倒计时，助你进入心流状态。
*   🎮 **解压乐园 (Stress Relief Games)**
    *   **极速打地鼠**：测试手速，释放压力。
    *   **无限捏泡泡**：强迫症福音，根本停不下来。
*   📊 **个人档案 (About Me)**
    *   自动统计日记篇数与任务完成度。
    *   记录你的“拾光”旅程。

## 🛠️ 技术栈 (Tech Stack)

*   **前端**: Vue 3 (Script Setup), Vite, Vue Router, Tailwind CSS
*   **后端**: Node.js, Express
*   **数据存储**: JSON File Adapter (轻量级文件存储，无需配置数据库)
*   **设计风格**: Neubrutalism / Pop Art

## 🚀 快速开始 (Quick Start)

### 1. 安装依赖
```bash
pnpm install
```

### 2. 启动开发服务器 (前后端同时启动)
```bash
pnpm run dev
```

*   **前端地址**: http://localhost:5174
*   **后端地址**: http://localhost:5175
